About me

Apa itu adaptive dan responsive web design? Apakah kamu setuju bahwa teknik tersebut bagus? Mengapa atau mengapa tidak?

Pada zaman zaman awal komputer dekstop, ukuran layar mereka besar, sehingga tidak terlalu mempertanyakan bagaimana tampilan kontent di browser ukuranya layarnya kecil seperti layaknya smart phone anda sekarang ini. Namun seiring dengan perkembangan teknologi, phone, smart phone, tablet bermunculan dan memiliki browser yang sama dengan yang ada di komputer desktop anda, tentu sangat sulit bila tampilan website di komputer dekstop sama dengan tampilan yang ada di smartphone anda, bila anda membukanya di smartphone anda, anda harus zoom / meperbesar dulu tampilan website biar bisa membaca apa isi dari website tersebut. maka dari itu orang orang berkumpul, mereka berdisukis bagaimana cara untuk mengatasi masalah ini, dan solusi atas permasalahan ini adalah responsive web design dan adaptive web design.

Responsive web design dan adaptive web design merukapan website yang menyesuaikan diri dengan lebih nyaman sesuai dengan ukuran lebar dari browser. mereka terlihat sama namun berbeda, perbedanya ada di cara kerjanya, dan html juga cssnya.

Responsive web design merupakan clinet-side, yang dimana artinya seluruh file website yang ada di server di kirim ke browser kita yang ada di komputer desktop, smart phone, tablet, dan berikutnya browser dari masing masing device merubah tampilan dari website berdasarkan lebar dari masing masing layar.

Adaptive web design merupakan server-side, yang dimana artinya ketika kita merequest suatu halaman website ke server, server mendikteksi terlebih dahulu device apa yang anda gunakan apakah itu komputer dekstop, smart phone, atau pun tablet. setelah server sudah tahu device apa yang anda gunakan, maka server mengirimkan file website yang sesuai dengan device yang sudah terditeksi.

Apakah hadirnya smartphone dan tablet mengubah desain web secara signifikan? Bagaimana kira-kira teknologi baru akan mengubah desain web di masa depan?

Ya tentu saja. ukuran komputer desktop dengan ukuran smart phone / tablet sangat lah berbeda jauh. bila tampilan website yang ada di dekstop di paksakan ke smart phone / tablet, tentu user akan kewalahan bahkan tidak mau membuka website tersebut. maka harus merubah desain web, dan di sesuaikan dengan ukuran device yang ada.

Bagaimana kira-kira orang yang memiliki kekurangan fisik menggunakan website? Apakah kita perlu menyesuaikan website untuk mereka juga? Solusi apa yang dapat mengatasi keadaan tersebut?

tergantung dari kekurangan fisik mereka, bila mereka tidak mempunyai jadi jari sulit bagi mereka untuk menguakan perangkat hardware seperti keyborad atau pun mouse, mungkin mereka bisa mengunakan kaki mereka untuk mengantikan hal tersebut, namun perkemangan teknologi dari sisi hardware dapat menjawab kekurangan fisik mereka tersebut. untuk masalah penyesuaian website, sepertinya tidak perlu, karena menurut saya, masalah yang dihadapi bagaimana orang orang yang memiliki kekurangan fisik dapat berinteraksi dengan hardware bukan dengan tampilan website. solusi untuk mengatasi orang orang yang memiliki kekurangan fisik adalah dengan cara membuat teknologi dari segi hardware(keyboard untuk kaki, dll) dan software (voice recognition)

Ide atau konsep apa yang sesuai denganmu saat mempelajari desain web?

saya mendapat ide dari melihat apa yang sudah ada sebelumnya, melihat banyak website di bidang sejenis dan berbeda, lalu mengkombinasikanya.
