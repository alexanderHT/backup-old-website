  1. Git dan GitHub :

    2. Sebagai web developer hal ini sangat sangat berguna karena fungsi dari Git untuk mengkolaborasikan satu project yang akan di kerjakan oleh banyak developer. permasalahan : saya punya project (misalkan project tersebut berbentuk folder dan isinya ada berberapa file(file1.text, file2.text, file3.text)) dan ada 3 developer yang sedang mengerjakan project tersebut, sebut saja si A, B, O namun ketiga developer ini tinggal saling berjauhan dan harus mengerjakan project yang sama, lalu bagaimana cara mereka biar bisa saling bergkolaborasikan project tersebut. sebelum ada github, mungkin mereka mengunakan email untuk menstranfer file file yang ada, misalkan si A membuat file baru file4.txt maka si A harus mengirim file tersebut mengunakan email ke si B dan O (repot), di tambah lagi bila jumlah developernya bertambah dan file yang di buat makin banyak, tentu mejadi sulit untuk di kontrol, maka dari itu muncul lah git dan github untuk mengatasi permasalahan di atas.

Bila ada penambahan, perubahan, pengahpusan yang terjadi di komputer masing masing hal tersebut hanya berefek pada komputer mereka, tapi tidak server, terkecuali mereka melakukan pengiriman folder dan file-file terbaru ke server maka folder dan file-file yang ada di server otomastis berubah. dan bila ada perubahan, mereka tinggal menarik lagi dari server.

    Cara mengunakanya:

    1. Install Git terlebih dahulu dari website GitHub

BLOG POST WEEK 2

Di minggu kedua hal hal yang di pelajari adalah dasar dasar pemerograman dengan mengunakan javascript, berikut hal hal yang sudah di pelajari:

  1. Tipe Data (Data type) :

    2. Data meruapakn fakta mentah, fakta yang mewakili sebuah objek, seperti pelanggan, karyawan, entitas, dan lain - lain yang di simpan dalam bentuk angka, huruf, simbol, teks, gambar, bunyi dan lain lain. sama halnya dengan komputer ketika kita memasukan fakta kedalam program, fakta fakta tersebut di simpan kedalam tipe data yang berbeda karena tipe data yang berbeda memiliki kateristik dan fungsi yang berbeda beda, berikut tipe data yang ada pada javascipt:

    • Angka (integer / Number): tipe data angka berfungsi untuk melakukan operasi matematika. seperti penjumlahan, pengurangan, perkalian, dan lain lain. contohnya: 1, 2, 3, .. dan seterusnya.
    • String: tipe data string berfungsi untuk menyimpan data dalam bentuk huruf-huruf dan tidak bisa melakukan operasi aritmatika. contohnya: "Alexander".
    • Booleans: benar atau salah (true or false) untuk menyatakan suatu kondisi benar atau salah. contoh: true / 1 dan false / 0.
    • Array: ketika tipe data number di masukan kesebuah tempat penyimpanan yang di sebut variable, maka variable itu hanya memiliki satu tipe data saja, dan hanya menyimpan satu nilai / value saja. dengan adanya array kita dapat menyimpan data lebih dari satu dengan tipe data yang berbeda- beda. contoh: var myArray = ["nama", 20, "Jakarta"]; .
    • Object: dengan object kita dapat menyimpan data - data dan fungsi secara bersamaan di satu tempat penyimpanan yang sama.

  2. Variable :

    3. merupakan sebuah sebuah tempat penyimpanan untuk menampung tipe data. contoh: var namaLengkap; . syntax var mengatakan kepada javascript untuk membuat variable ( tempat penyimpanan ) dengan nama variable setelah syntax var yaitu namaLengkap.


  3. Function :

    4. Sebuah kumpulan statement yang lebih dari satu di bungkus menjadi satu, agar nantinya ketika ingin di gunakan lagi maka tinggal di panggil saja. sebuah function juga dapat di beri inputan yang di sebut parameter.

    contohnya :

    var namaFunction = function(parameter){

    // steatment satu

    // steatment dua

    }

  4. Selection / Memilih kondisi :

    5. kita memberi tahukan kepada komputer bahwa terkadang ada sebuah steatment program yang harus di jalan kan dalam kondisi tertentu (true and false), sehingga bila komputer menemukan kondisi yang sesuai maka komputer akan mengeksekusi steatmen tersebut. untuk selection ada berberapa jenis yaitu:

    • if : jika statement ini di jalankan maka, steatment ini akan menjalakan statement yang ada di dalam bracket ( { statement } ) jika kondisi yang ada di parameter true

      Syntax

      if(condition){

      // do this

      }

    • if else : jika statement ini di jalankan maka, steatment ini akan menjalakan statement yang ada di dalam bracket ( { statement } ) jika kondisi yang ada di parameter true, bila kondisinya salah maka akan menjalakan statement yang ada di dalam else

      Syntax

      if(condition){

      // do this

      }else{

      // if condition false so do this

      }

    • if / else if / else : jika statement ini di jalankan maka, steatment ini akan menjalakan statement yang ada di dalam bracket ( { statement } ) jika kondisi yang ada di parameter true, bila kondisinya salah maka akan menjalakan statement yang ada di bawahnya ya itu else if, bila kondisinya benar maka akan menjalankan statement yang ada didalamnya, bila salah maka lanjut ke berikutnya

      Syntax

      if(condition){

      // do this

      }else if(condition){

      // if the condition abouv is false so do this

      }else{

      // if condition false so do this

      }

  5. loop / perulangan :

    6. Loop menjalakan intruksi atau steatment secara berulang ulang sampai menemukan kondisi yang di tentukan. fungsinya agar kita dapat mengunakan fungsi atau steatment secara berulang ulang sampai kondisi yang di tentukan tercapai. ada berberapa macam loop yaitu: 1. while - loops through a block of code as long as the specified condition is true 2. do...while - loops through a block of code once, and then repeats the loop as long as the specified condition is true 3. for - loops through a block of code a specified number of times 4. foreach - loops through a block of code for each element in an array

    • FOR : Tipe perulangan ini melakukan perulangan dengan spesifik waktu yang sudah di tentukan

      • Contoh :

      for(var i = 0 => start; i < 10 => end; i++ => increment or decrement){

      // do steatment

      }

    • WHILE : Tipe perulangan ini melakukan perulangan dengan ketentuan bila kondisinya benar, bila kondisinya salah maka perulangan akan terhenti

      • Contoh :

      condition = true;

      while(condition){

      // do statement

      // condition = false;

      }

    • DO WHILE : Tipe perulangan ini sama degan WHILE namun bedanya statement yang ada mau di ulang di jalankan terlebih dahulu minimal 1 kali

      • Contoh :

      do{

      // do statement

      // condition = false;

      }while(condition);

    • FOR EACH : Tipe perulangan ini melakukan perulangan terhadap array atau juga object, akan mengulang sebanyak isi yang ada didalam array tersebut

      • Contoh :

      var array = [1, 2, 3, 4];

      for(var a in array){

      // do statement

      // console.log(a);

      }

  6. Object :

    7. Object dalam bahasa pemerograman merupakan suatu objek yang ada di dunia nyata di tiru dan dibuat kedalam virtual (objek virutal). yang di tiru adalah infromasi (nama, umur, warna, tinggi, dll) dan juga sifat sifat (makan, berjalan, dll) dari objek tersebut, objek merupakan sebuah tipe data yang menyimpan property (informasi/ data) dan juga method (sifat / fungsi) yang di simpan menjadi satu di dalam objek. pembuatan objek ada dua yaitu:

    • Object Literal Notation :

      • Contoh :

      var me = {

      name : 'Alexander',

      age : 21

      };

      contoh di atas merukapan syntax untuk membuat sebuah objek bernama me dan me memiliki property nama dan umur

    • Object Constructor :

      • Contoh :

      var me = new Object();

      me.name = "Alexander";

      me.age = 21;

      contoh di atas merukapan syntax untuk membuat sebuah objek bernama me dan me memiliki property nama dan umur

  7. constructor / class :

    8. constructor atau class merupakan blue print dari sebuah object, kategori dari objek yang akan dibuat, cetakan dari objek. fungsinya agar mempermudah/ mempersimpel dan tidak di ulang ulang hal yang sama dalam pembuatan objek.

    Contoh :

    ================ Class ==================

    function Person(job, married) {

    // property

    this.job = job;

    this.married = married;

    // method speak

    this.speak = function(){

    console.log("Hello!");

    }

    }

    ================ Objek ===================

    var me = new Person("Programmer", false);

    ================ Access ==================

    console.log(me.job);

    console.log(me.married);

    me.speak();
